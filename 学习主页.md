## 最近编辑的笔记

```dataview
TABLE WITHOUT ID
  file.link AS "笔记",
  dateformat(file.mtime, "MM-dd HH:mm") AS "修改时间"
FROM "solutions" OR "advanced-math"
WHERE file.name != "学习主页"
SORT file.mtime DESC
LIMIT 10
```

---

## 各章节笔记数量

```dataview
TABLE WITHOUT ID
  rows.file.folder AS "目录",
  length(rows) AS "笔记数"
FROM "solutions" OR "advanced-math"
WHERE file.name != "学习主页"
GROUP BY file.folder
SORT length(rows) DESC
```

## 待办任务（来自所有笔记）

TASK

## 标记为待复习 / 存疑的笔记

```dataview
LIST
FROM "solutions" OR "advanced-math"
WHERE contains(file.name, "存疑") OR contains(tags, "复习") OR contains(tags, "待重做")
SORT file.mtime DESC
```

> [!tip] 用法
> 想让某篇笔记出现在"待复习"列表里，在它的属性区加 `tags: 复习` 即可。间隔复习卡片由 Spaced Repetition 插件单独管理（命令面板搜 "Review"）。
